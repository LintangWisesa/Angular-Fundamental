## ‚≠ê Getting Started

- ### Install ```angular-cli``` package

    ```
    $ npm i -g @angular/cli
    ```

- ### Check Angular version

    ```
    $ ng --version
    ```

- ### Initialize a new Angular project

    ```
    $ ng new yourProjectName
    // routing? yes
    // styling? CSS
    // share data with Google? no
    ```

- ### Running a project

    ```
    $ cd yourProjectName
	$ ng serve --open
    ```
